# 溪水波浪特效测试

Godot 4.7 test project copied from the first creek level.

Open this folder as a Godot project and run `Main.tscn`.

- Arrow keys: move the player word `我`
- `Space`: interact with the word in front
- `Backspace`: delete a deletable word in front
- `Tab`: split a splittable word in front
- `Alt` + arrow keys: pull a pushable word
- `Esc`: quit the test project

The test starts on the copied gem level and triggers the effect automatically after a short delay.
