# 文字游戏人工复核台使用说明

打开 `review.html` 开始复核。

## 复核范围

- 视频事件：确认整章原视频时间点、帧号、事件描述。
- 截图语义：确认截图状态、玩家格子、可见文字和动态对象。
- 源码归属：不进入人工复核，由 AI 自动分析。

## 头盔视频规则

- 使用整章原视频：`参考资料/视频参考/四目头盔视频 .mp4`。
- 头盔所有时间点都按整章视频原始时间轴记录。
- 只处理 `36:50-51:00` 这一段，不换算片段内时间码。

## 操作方式

1. 左侧筛选视频事件或截图语义。
2. 右侧核对原图、来源路径和 AI 预填字段。
3. 内容正确就点“确认”。
4. 内容需要修正就改字段后点“保存修改”，或点“需要修改”并写备注。
5. 复核完成后点“导出审核结果”，得到 `wordgame_review_result.json`。

浏览器会把复核状态存在本机 `localStorage`。导出结果不会自动修改 CSV，需要后续统一导入。
